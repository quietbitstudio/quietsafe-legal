QuietSafe Privacy Policy

Publisher: QuietBit Studio

Contact: quietbitstudio@gmail.com

Effective date: 2026-06-16

Summary

QuietSafe is designed as a local encrypted vault application. Vault contents are stored locally on the user's device and are not uploaded to a QuietSafe cloud service.

Data processed by the application

QuietSafe may process vault names, vault metadata, encrypted vault files, imported file names and contents, preview data, application settings, and temporary runtime data required for safe preview or playback workflows.

Data collection and transmission

QuietSafe does not intentionally collect, transmit, sell, or share vault contents, passwords, encryption keys, imported files, or exported files with the publisher.

Passwords and encryption keys

QuietSafe does not provide publisher-side password recovery. If the master password is lost, encrypted vault data may be permanently unrecoverable.

Local storage

QuietSafe stores vault data on the user's device in user-selected or application-managed folders. The user is responsible for backups and for protecting device access.

Temporary files

QuietSafe is designed to avoid unnecessary full decrypted copies of protected content. Some operations may still require temporary runtime data for previews, media playback, export, or operating system integration. Temporary data should be removed when no longer needed.

Exported files

Files exported from QuietSafe are outside the protection of the encrypted vault. The user is responsible for managing and deleting exported files.

Support communications

If the user contacts support, QuietBit Studio may receive information voluntarily provided by the user, such as email address, screenshots, logs, or a description of a problem. Users should not send vault passwords or sensitive files to support.

Website analytics and advertising measurement

The QuietBit Studio website uses Microsoft Advertising Universal Event Tracking (UET), tag ID 187262760, to measure website visits, Microsoft Store link clicks, and advertising campaign performance. This website starts UET with advertising storage denied. If a visitor selects “Allow all”, the consent state changes to granted and Microsoft Advertising may use advertising cookies. If a visitor selects “Necessary only”, UET remains in a limited mode and may still send cookieless measurement signals.

The visitor's consent choice is stored in the browser's local storage and can be changed at any time with the “Privacy settings” button. For more information about Microsoft's processing, see the Microsoft Privacy Statement.

Children

QuietSafe is not intended to knowingly collect personal information from children.
